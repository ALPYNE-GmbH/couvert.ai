Gmail — Schritt für Schritt

1. Signatur kopieren

  1. Oben bei Variante A (oder B) den roten Button Kopieren klicken.
  2. Der Button wird kurz grün ✓ Kopiert. Die Signatur liegt jetzt exakt und ohne Einzug auf deiner Zwischenablage.

2. Alte Signatur in Gmail löschen

  1. Gmail öffnen. Oben rechts Zahnrad → Alle Einstellungen ansehen.
  2. Tab Allgemein → runter scrollen zu Signatur.
  3. Falls noch eine alte Signatur drin steht: ins Signatur-Textfeld klicken, Cmd + A drücken (alles markieren), Entf (alles löschen). Das Feld muss komplett leer sein.
  4. Falls du mehrere Adressen in Gmail hast (hello@alpynelab.com separat): oben bei Signatur auf + Neu erstellen klicken, Namen z. B. "Alpyne", dann das Feld leer lassen.

Dieser Schritt ist wichtig: Reste alter Einfügeversuche (versteckte `<div>`-Wrapper) verursachen den Einzug, den du bei der Couvert-Signatur gesehen hast.

3. Neue Signatur einfügen

  1. Cursor ins leere Signatur-Feld setzen.
  2. Cmd + V (Mac) oder Ctrl + V (Windows).
  3. Die Signatur erscheint direkt gerendert — Wortmarke, Schweizer Kreuz, Tagline, Datenblock.
  4. Unten bei Standardwerte für Signatur die neue auswählen für Neue E-Mails und Antworten/Weiterleitungen.
  5. Falls du Absender-spezifische Signaturen nutzt: stelle sicher, dass die Alpyne-Signatur mit hello@alpynelab.com verknüpft ist, nicht mit max@couvert.ai.
  6. Ganz unten Änderungen speichern.

4. Test

  1. Neue E-Mail öffnen, oben Absender auf hello@alpynelab.com umstellen.
  2. Die Alpyne-Signatur sollte erscheinen, am linken Rand bündig mit deinem Fliesstext.
  3. Probemail an dich selbst schicken — so siehst du, wie die Empfänger die Signatur bekommen.

Warum der HTML-Code allein nicht reicht

Gmail hat keinen HTML-Quellcode-Modus für die Signatur. Wenn du den Code <table>...</table> direkt ins Signatur-Feld kopierst, bleibt er als Text stehen, weil das Feld ein WYSIWYG-Editor ist, kein Code-Editor. Der Kopieren-Button oben löst das: er legt die Signatur als rendered HTML auf die Zwischenablage (nicht als Text), und Gmail übernimmt beim Einfügen Layout plus Schweizer Kreuz direkt.

Apple Mail

  1. Mail → Einstellungen → Tab Signaturen.
  2. Links deinen Alpyne-Account auswählen, + für neue Signatur. Namen vergeben (z. B. "Alpyne").
  3. Rechts im Textfeld den Standard-Text markieren und löschen.
  4. Kopieren-Button oben klicken, dann zurück in Apple Mail, ins leere Feld klicken und Cmd + V.
  5. Falls das Häkchen "Immer meinen Standardschriftschnitt verwenden" aktiviert ist — wegklicken. Sonst überschreibt Apple Mail die Formatierung der Signatur.
  6. Einstellungsfenster schliessen. Neue Mail öffnen, Signatur sollte automatisch erscheinen.

Outlook (Web oder Desktop)

  1. Outlook Web: Zahnrad → Alle Outlook-Einstellungen anzeigen → E-Mail → Verfassen und Antworten.
  2. Bei E-Mail-Signatur einen Namen vergeben, ins Feld klicken, Kopieren-Button hier oben drücken, dann Cmd + V / Ctrl + V.
  3. Speichern. Outlook Desktop: derselbe Weg über Einstellungen → Signaturen.

Hinweis: Das Schweizer Kreuz steckt als base64-PNG direkt im HTML — das wird in Gmail, Apple Mail und Outlook (Web + Desktop) korrekt gerendert. Kein externes Laden, keine Sicherheits-Filter, die es entfernen würden.
